Project for @CodSoft Python Internship 

A functional command-line calculator that performs basic arithmetic operations based on user input.

Key Features:


Arithmetic Operations: Supports Addition, Subtraction, Multiplication, and Division.


User-Friendly Input: Prompts users for two numbers and an operator choice.

Error Handling: Includes checks for division by zero and invalid numeric inputs.

Run Command: python task2_calculator.py
